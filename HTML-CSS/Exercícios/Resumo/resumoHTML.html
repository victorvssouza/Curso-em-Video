HTML

HTML → HyperText Markup Language
• É usado para criar a estrutura de um site;


Princípio das Tags

Há muitas Tags que apenas mudam o estilo do texto sem adicionar nenhum significado, mas de acordo com o W3C o que for forma fica no CSS e o que for sentido fica no HTML.


Tipos de Tags

Tags de Título:

<h1>Titulo</h1>

<h2>Titulo</h2>

<h3>Titulo</h3>

<h4>Titulo</h4>

<h5>Titulo</h5>
<h6>Titulo</h6>

Tags de Texto:

• <strong>Negrito<strong>

• <em>Ênfase<em>

• <mark>Texto marcado</mark>

• <cite>Citação de uma obra</cite>

• <del>Edição, Modificação, Tachado</del>

• <small>Texto Pequeno</small>

• <big>Texto Grande</big>

• <ins>Texto Inserido</ins>

• <sup>Texto Sobrescrito</sup>

Para inserir coisas do tipo x20+3

• <sub>Texto Subcrito</sub>

Para inserir coisas do tipo H2O

Tags de Parágrafo e Quebra de Linha:

<div>
<p>Parágrafo dentro de container de fluxo de conteúdo</p>
</div>

<p>

Parágrafo

</p>

<br>
Quebra de Linha

<hr>

Quebra Temática

A maioria das Tags precisa ser aberta e fechada para limitar até onde deve influênciar.
Ex: <h1></h1>
Algumas só precisam ser abertas, pois sua influência não alcança outras linhas do código.
Ex: <br>
